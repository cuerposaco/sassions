1. Subir el proyecto al repositiorio GIT

El repositorio debe estar creado previamente

 
		> git init
		> git remote add origin [git://url_repositorio_git.git]
		> git add --all
		> git commit -am "Release"
		> git push -u origin master

2. Publicar en npm [Sinopia] versionando el modulo

 
		> npm version [ patch | minor | release]
		> npm publish

3. Publicar en bower [Private Bower] 


		> bower register [nombre_del_modulo] [http://url_repositorio_git]

private bower actúa como un proxy contra el repositorio git, recogiendo los tags de su repositorio

3. Subir los cambios y tagear la nueva version en el repositorio GIT


		> git push --follow-tags

éste comando recoge la version del package.json usado por npm y la utiliza para tagear el commit
