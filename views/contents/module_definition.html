¿Qué es un módulo?

Un módulo es un elemento de la página que tiene un comportamiento propio.

Debe tener la carácterística de que en la misma página puedan existir varios módulos del mismo tipo funcionando de forma independiente o interoperando entre ellos sin conflicto.

Deberían ser instancias únicas



		// Ejemplo de un módulo
		function HelloModule(opts){
			if( !opts.el ){
				return;
			}
			this.el = opts.el;
			this.el.innerHTML = this.geeting();
		}

		HelloModule.prototype.greeting = function(){
			return 'Hello World!!!';
		}

Instanciación de un Módulo


		// Instanciación de un módulo
		var greetingNodes = document.querySelectorAll( '.greeting_module' );
		for( var i = 0; i < greetingNodes.length; i++ ){ new HelloModule({ el: greetingNodes[i] }); }
